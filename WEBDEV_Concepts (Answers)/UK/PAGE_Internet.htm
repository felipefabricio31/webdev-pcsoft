This page is intended to be displayed on Internet.
A fixed URL (fixed address at which the site can be contacted at any time) is required in order for this page to be properly indexed by the search engines.
Then, all you have to do is include content in the page in order for the search engines to index the content.
The search engines operate in "ramification", which means that the webmaster of a site gives the address of the first page of his site to the search engines.
Then, the search engine will display this page and analyze it.
1 - It will index its content
2 - It will retrieve the list of links in order to index the pages toward which the page is pointing.
